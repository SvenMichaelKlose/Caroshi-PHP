Neue Records erstellen

record_create() erzeugt einen neuen Record im Source des Context-Cursors oder anhand eines als Argument übergebenen source sets, welches das erstellen von Records in verschiedenen Sources inklusive Referenzen zwischen den Records erlaubt.
