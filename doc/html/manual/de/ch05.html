Widget-Set-Funktionen

admin_panel bietet Widget-Funktionen zum erstellen von Dokumenten zusammen mit Tabellen-Funktionen zur Formatierung (gridbag layout). Der Inhalt der Elemente und die wichtigsten Attribute können als Funktionsparameter übergeben werden. BSP. Damit dies funktioniert müssen URLs mit der Function admin_panel::url() anstatt mit application::link() erstellt werden.
