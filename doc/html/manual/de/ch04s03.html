Event-Argumente

Generell können Events beliebig viele Argumente für den Empfänger enthalten. Diese werden bevorzugt zu beginn eines Handlers mit der Methode application::arg() abgerufen. Sie erwartet als ersten Parameter den Namen des Arguments gefolgt von optionalen Flags die das Verhalten der Methode beeinflussen. So bricht die Funktion den Programmablauf ab, wenn ein Argument nicht im Event vorkommt, es sei denn das Flag ARG_OPTIONAL wurde gesetzt.
