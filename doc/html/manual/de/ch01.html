Chapter 1. Einleitung

Caroshi [1] (vormals dev/con php base) ist eine objektorientierte Bibliothek zum entwickeln von PHP Web-Applikationen im traditionellen Programmierstil. Sie enthält neben allgemein verwendbaren Funktionen und Datenbankschnittstellen einen event-orientierten Application-Server und ein darauf aufbauendes (HTML-User-Interface).
