Mischen von Cursors verschiedener Typen

cursor_merged-Objekte bieten die Möglichkeit meherere Cursor-Objekte zu einem einzigen zusammenzufassen. Die Cursors werden der Methode query() als Array übergeben. Beim auslesen wechselt get() zum nächsten Cursor, wenn der aktuelle vollständig ausgelesen wurde. Aufrufe von key(), field(), source() und set_field() werden an den zuletzt ausgelesenen Cursor weitergeleitet. set_source(), set_key(), create() und delete() können nicht verwendet werden.
