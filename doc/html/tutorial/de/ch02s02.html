defaultview ()

Wurde keine aufzurufende Funktion oder Methode in der URL angegeben, wird stets die Funktion defaultview () in der abgeleiteten Klasse aufgerufen. Eine per URL aufrufbare Funktion kommt die Aufgabe der Seitengenerierung zu und wird 'view' bezeichnet.
